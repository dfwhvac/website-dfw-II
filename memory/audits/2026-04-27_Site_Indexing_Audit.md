## How to use this file

For each URL below, run **GSC → URL Inspection** (paste URL → press Enter). Mark the result:

- `[✅ Indexed]` — "URL is on Google" — done
- `[❌ Not indexed]` — "URL is not on Google" — submit via "Request Indexing" if budget remains
- `[🔄 Submitted]` — submitted but not yet confirmed indexed — recheck in 3–7 days
- `[➖ Skip]` — intentionally excluded (e.g. /studio admin, redirect-source legacy URLs)

**Daily Request Indexing budget:** ~10/day per property. Don't waste slots on already-indexed URLs.
